
# Badminton ERP Pro — Next.js + Tailwind + shadcn (minimal) + Supabase

This project migrates your Google Apps Script queue system to a modern web app.

## Quick start
1) Create a Supabase project → copy the URL & keys
2) Run SQL in `supabase/schema.sql` (SQL Editor)
3) Copy `.env.example` to `.env.local` and fill keys
4) Run:
```bash
npm install
npm run dev
```

Open http://localhost:3000

**Admin PIN**: set via `ADMIN_PIN` (default `kpmgadmin`).
**Auto refresh**: `NEXT_PUBLIC_AUTO_REFRESH_MS` (ms).
